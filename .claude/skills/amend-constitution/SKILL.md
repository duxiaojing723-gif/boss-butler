# /amend-constitution

## 触发方式

```
/amend-constitution
修改原因：[为什么要改宪法]
具体修改：[哪条规则，改成什么]
```

## 执行流程

### Step 1：评估必要性
先问这三个问题：
1. 这次改动是因为宪法规则本身有问题，还是因为当前 task 不方便？
2. 如果是"当前 task 不方便"，答案是"不该改宪法，应该调整 task 范围"
3. 如果宪法规则确实有问题，继续

### Step 2：创建变更决策记录
在 `docs/decisions/constitution-changes.md` 追加：
```markdown
## 变更 [日期]
**修改内容**：[具体改了什么]
**原因**：[为什么要改]
**风险**：[这次修改可能带来什么副作用]
```

### Step 3：修改 CLAUDE.md
在记录完成后才允许修改 `CLAUDE.md`。
修改完成后更新"宪法版本"部分。

### Step 4：提醒
告知用户：宪法已更新，后续所有 task 将按新规则执行。
